# Frontend da Aplicação SISLEGIS

## Download das dependências através do Bower

* Instalar o [npm](https://www.npmjs.org)
```
$ sudo yum install npm
```
* Instalar o [bower](http://bower.io/)
```
$ sudo npm install
```
* Baixar as dependências
```
$ bower install
```
* Executar tasks
```
$ gulp
```
* Executar tasks (watch)
```
$ gulp
```
